# d-001

# D Installation

```bash
$ sudo apt install ldc dub
```

# Initialize the sample project:

```bash
$ dub init d-001
```

* This will create d-001 folder.

# Running the project

```bash
$ dub run
```

# Adding gtk-d

```bash
dub add gtk-d
```

# Executing the result application

```bash
$ ./d-001
```
